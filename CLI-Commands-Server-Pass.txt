üîë Step 1. Wallet (Identity & Funding)
npm run prod-wallet


Generates/loads your wallet (CFG/ETH address).

Fund this wallet with native tokens (for gas) and stablecoins (for repayments).

This is your identity across assets & pools.

üè¶ Step 2. Pools (Where borrowing happens)
npm run prod-pools


Lists available credit pools you can join.

Pick the pool that matches your RWA type (invoice, real estate, etc.).

Each pool has: PoolId, currency (DAI/USDT/USDC), LTV policy, APR.

Note your PoolId ‚Äî you‚Äôll need it to pledge/borrow.

üìú Step 3. Assets (Tokenize your RWA)
npm run prod-assets


Here you mint an NFT that represents your real-world asset (invoice, property, contract, etc.).

Input: JSON metadata (like asset.json we drafted earlier).

Output: AssetId (your token on-chain).
Once minted, this NFT can be pledged as collateral.

üìå Step 4. Borrow (Pledge & Draw Credit)
npm run prod-borrow


First: pledge your AssetId into the chosen PoolId.

The CLI should show max borrowable amount = FaceValue * LTV.

Then: request drawdown (borrow X amount).

Funds are sent to your wallet (DAI/USDT/USDC).

üìä Step 5. Portfolio (Track Loans & Collateral)
npm run prod-portfolio


Monitor outstanding loans, pledged assets, repayment schedules.

Repay loans here as well.

When fully repaid ‚Üí release NFT collateral back to your wallet.

prod-borrow ‚Üí pledge + draw credit

prod-portfolio ‚Üí manage, repay, release
